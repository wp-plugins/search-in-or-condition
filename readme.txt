Adjust your conditional operators into the search query

== Description ==

wp-search-inorcondition allows you easily to set the "OR" condition for "every single word" of your research "s" querystring value

== Installation ==
1. Extract the wp-search-inorcondition/ folder file to /wp-content/plugins/
2. Activate the plugin at your blog's Admin -> Plugins screen
